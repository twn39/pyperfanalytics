# PyPerfAnalytics API 文档

本项目根据不同的功能模块划分为以下几个详细的 API 文档。每个文档都包含了函数的明确输入参数、返回参数以及相应的代码示例。

## 模块列表

- [Returns 模块](returns.md): 提供 returns 相关的计算或图表功能
- [Risk 模块](risk.md): 提供 risk 相关的计算或图表功能
- [Drawdowns 模块](drawdowns.md): 提供 drawdowns 相关的计算或图表功能
- [Tables 模块](tables.md): 提供 tables 相关的计算或图表功能
- [Charts 模块](charts.md): 提供 charts 相关的计算或图表功能
- [Utils 模块](utils.md): 提供 utils 相关的计算或图表功能
